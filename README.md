# kira-ai-plugin-qq-enhance

增强数字生命在 QQ 平台的能力，提供贴表情回应、资料卡点赞、撤回消息、禁言群成员、表情控制和私聊输入中状态等扩展功能。

## 功能

### 贴表情

AI 可以对收到的消息自主回应 QQ 表情（类似长按消息贴表情）。

- 默认**启用**
- 调用底层接口：`set_msg_emoji_like`

### 资料卡点赞

AI 可以给 QQ 用户的资料卡点赞。

- 默认**关闭**，需手动在配置中启用
- 单次最多点赞 50 次，每次以 10 个为单位分批发送
- 调用底层接口：`send_like`

> Tips：此功能可能由于QQ限制导致不可用

### 撤回消息

AI 可以撤回自己或群成员的消息（如果有权限）

- 默认**启用**
- 调用底层接口：`delete_msg`

### 禁言群成员

AI 可以禁言群成员（如果有权限）

- 默认**启用**
- 调用底层接口：`set_group_ban`

### 表情控制（新增）

AI 发送表情（Sticker）时，可以控制保留概率和插入位置。

- 默认**启用**
- **保留概率**：每个表情被保留的概率（0~1），低于概率的会被删除
- **随机位置**：表情可以随机插入到消息中间，而不是固定在末尾
- 自动过滤只有引用的消息块，提升回复质量
- 底层接口：本地处理，无外部接口调用

### 私聊输入指示（新增）

在私聊中自动显示“正在输入...”状态，提升交互体验。

- 默认**启用**
- **首次延时**：收到消息后延迟指定秒数显示“正在输入...”
- **持续发送**：每隔固定时间重新发送一次，直到 AI 完成回复（无工具调用）
- 仅对私聊生效，群聊自动禁用
- 调用底层接口：`set_input_status`（注意：napcat有而LLonebot无此接口）

## 配置

在插件配置页面可调整以下选项：

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 启用贴表情 | AI 是否可以对消息回应表情 | 开启 |
| 启用资料卡点赞 | AI 是否可以给用户资料卡点赞 | 关闭 |
| 启用撤回功能 | AI 是否可以撤回消息 | 开启 |
| 启用禁言功能 | AI 是否可以禁言群成员 | 开启 |
| 启用表情控制 | 是否控制表情发送概率和位置 | 开启 |
| 表情保留概率 | 每个表情被保留的概率（0~1） | 0.5 |
| 随机位置 | 表情是否随机插入到消息中间 | 开启 |
| 启用私聊输入指示 | 是否在私聊显示“正在输入...” | 开启 |
| 首次延时（秒） | 显示“正在输入...”前的延迟 | 2.0 |
| 持续发送间隔（秒） | 重复发送“正在输入...”的间隔 | 2.0 |
| 工具说明 | 额外提示词说明工具使用场景 | ... |

## 依赖

- 仅支持 **QQ 平台**适配器（`platform == "QQ"`）
- 需要底层客户端支持 NapCat / OneBot 相关接口
